# FlappyBirds

Flappy Birds in C and OpenGL.

## Dependencies

- [GLFW](https://www.glfw.org/)

### Install dependencies (Ubuntu)

```bash
sudo apt install libglfw3-dev
```

## Build (Linux only)

```bash
make -j`nproc`
```

## Run

```bash
./flappy_birds
```

## Controls

- Space: Jump
